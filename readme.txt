** TINKER BELL ~ FAIRY CATCH **
Author: Kelly Chen
Version: 1.0

Objective: To hit the Neverbeast with the flower as fast as possible

Gameplay:
- Use the UP, DOWN, LEFT, RIGHT controls to move Tinker Bell up/down and right/left on the screen
- Press A (Z on keyboard) to release the flower
- Press SELECT to exit the game
- Score will decrease as time goes on
